Math for Data Science Review

This article is based on Coursera course Math for Data Science provided by Duke University. To better understanding the algorithms in data mining, machine learning and deep learning, I write this summary to go through the basic math that will be used.

Sets

A set means a collection of some elements. For example, a basket of vegetables.

Cardinality is the size of a set. It means the number of elements in the set. For example, if A = {1,2,3,4}, $|A|=4$.

Intersection of A and B, $ A\bigcap B $,is a set that contains the element that occurs in both set A and B.

Union of A and B, $ A \bigcup B$, is a set that contains all elements from set A and B.
